MENSAJES DE ERROR

 

Mensaje N° 1

Este mensaje se visualiza cuando el usuario ha ingresado una contraseña incorrecta para acceder al sistema. Al aceptar la opción le permitirá ingresar una nueva contraseña. Solo puede equivocarse en tres oportunidades caso contrario el sistema se cerrará automáticamente.

 

 

 

Mensaje N° 2

                                                       

 

 

Este mensaje se visualiza cuando el usuario intenta dejar vacío un campo. Ningún campo  puede quedar vacio.
